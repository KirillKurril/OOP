# Длинные нарды
##### Проект представляет собой оффлайн десктопную версию игру в длинные нарды.
##### В качестве языка для написания основополагающей библиотеки игры используется C# и .net framework 8.0.
##### Библиотека содержит четыре класса: Game, GameBoard, Piece и Player, реализующих основную бизнес-логику приложения. ##### Публичным классом является только класс Game. Взаимодействие внешней части приложения с внутренними абстракциями осуществляется через его методы. Ниже предосталена UML-диаграмма классов основной бмблмотеки приложения.

#####В качестве фреймворка для создания пользовательского интерфейса будет использован Unity.
#####При успешных выполнениях поставленных задач, в последующих патчах планируется использовать PostgreSQL.

Для ознакомления с правилами игры в длинные классические нарды вы можете посетить следующую ссылку: "Как правильно играть в длинные классические нарды" (https://h-present.ru/blog/kak-pravilno-igrat-v-dlinnye-klassicheskie-nardy/).
